## Build Setup

``` bash
# install dependencies
cnpm install

# serve with hot reload at localhost:3001
cnpm run demo

# serve with express at localhost:3000
cnpm run api
```

## Notice

> 如果想要验证用fetch获取数据，需先运行cnpm run api打开本地express创建的临时接口，获取数据
